# www.toyolab.io
Website of ToyoLab
Add Event Component
## Development Guide
1. clone this repository via 
  ```shell
  git clone https://github.com/toyolabio/www.toyolab.io.git
  ```
2. cd into www.toyolab.io
  ```shell
  cd www.toyolab.io
  ```
3. Check into Development branch
  ```shell
  git checkout development
  ```
4. Install nodemodule via
  ```shell
  npm install
  ```
  
  or (recommended)
  
  ```shell
  yarn
  ```
  
5. Run project

  ```shell
  yarn develop
  ```
  
  or 
  
  ```shell
  npm run develop
  ```
