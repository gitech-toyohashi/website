# www.toyolab.io
Website of ToyoLab

##Development Guide
1. clone this repository via 
  ```shell
  git clone https://github.com/toyolabio/www.toyolab.io.git
  ```
2. Check into Development branch
  ```shell
  git checkout development
  ```
3. Install nodemodule via
  ```shell
  npm install
  ```
  
  or (recommended)
  
  ```shell
  yarn
  ```
  
4. Run project

  ```shell
  yarn develop
  ```
  
  or 
  
  ```shell
  npm run develop
  ```
